# InternHub – Full Stack Internship Management System

## 📌 Project Overview

InternHub is a Full Stack Internship Management System developed using Java Spring Boot, MySQL, HTML, CSS, and JavaScript. It provides a platform where students can explore internships, apply online, track their applications, and analyze their internship readiness, while administrators and companies can efficiently manage internships and applicants.

---

## 🚀 Features

### 👨‍🎓 Student Module

* Student Registration & Login
* Student Profile Management
* Internship Readiness Score
* Failure Analyzer
* Internship Journey Timeline
* View Available Internships
* Apply for Internships
* Track Application Status

### 🏢 Company Module

* Company Registration & Login
* Company Dashboard
* Post New Internships
* Edit/Delete Internships
* View Applicants

### 👨‍💼 Admin Module

* Admin Login
* Manage Students
* Manage Companies
* Manage Internships
* View Applications
* Approve/Reject Applications

## ▶️ How to Run

1. Clone the repository.
2. Open the Backend project in your IDE.
3. Configure MySQL in `application.properties`.
4. Run the Spring Boot application.
5. Open the Frontend using Live Server.
6. Register/Login and explore the application.
