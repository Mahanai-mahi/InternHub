# InternHub - Internship Management System

## Overview

InternHub is a Full Stack Internship Management System developed using Java Spring Boot, MySQL, HTML, CSS, and JavaScript. It provides separate portals for Students, Companies, and Admins to simplify the internship recruitment process.

---

## Technologies Used

### Backend

* Java
* Spring Boot
* Spring Data JPA
* Maven

### Frontend

* HTML5
* CSS3
* JavaScript
* Font Awesome

### Database

* MySQL

## How to Run

1. Clone the repository.
2. Open the backend in your IDE.
3. Configure MySQL database.
4. Run the Spring Boot application.
5. Open the frontend using Live Server.
6. Access the application.
